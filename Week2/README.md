# Week 2 - Introduction to Data Analysis with NumPy and Pandas

## Overview

This week focuses on learning the fundamental tools used in Data Science for data manipulation, analysis, and preprocessing. The main objective is to understand how data is stored, analyzed, cleaned, and prepared for further use in Machine Learning and Data Science projects.

---

## Topics Covered

### Day 1
- Introduction to NumPy
- Why NumPy is used
- Creating Arrays
- Array Properties

### Day 2
- NumPy Array Operations
- Mathematical Operations
- Indexing and Slicing
- Working with Array Elements

### Day 3
- Introduction to Pandas
- Series and DataFrames
- Creating DataFrames
- Basic Data Exploration

### Day 4
- Working with CSV Files
- Reading CSV Files
- Writing CSV Files
- Exploring Dataset Structure

### Day 5
- Basic Data Cleaning
- Handling Missing Values
- Removing Duplicates
- Data Filtering
- Selecting Specific Rows and Columns

### Day 6
- Mini Project
- Applying NumPy and Pandas Concepts
- Data Analysis Workflow

---

## Skills Gained

By completing this week, I will be able to:

- Create and manipulate NumPy arrays
- Perform numerical operations efficiently
- Work with Pandas DataFrames
- Read and analyze CSV datasets
- Clean and filter data
- Prepare datasets for analysis
- Apply data analysis concepts in a mini project
