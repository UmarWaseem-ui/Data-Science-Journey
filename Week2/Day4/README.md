# Day 4 - Reading and Exploring CSV Files

## Objective

To learn how to load datasets from CSV files using Pandas and explore their structure and contents before performing data analysis.

---

## Topics Covered

### Reading CSV Files

* Loading datasets using Pandas
* Understanding the structure of CSV files

### Dataset Exploration

* Viewing the first and last records
* Understanding the size and structure of a dataset
* Exploring column names and data types
* Generating summary statistics

---

* `pd.read_csv()`
* `head()`
* `tail()`

* `columns`

* `describe()`

## Key Learnings

* CSV files are one of the most common formats for storing datasets in Data Science.
* Pandas provides an easy way to load and analyze CSV files.
* Exploring a dataset is an essential first step before cleaning, visualizing, or modeling the data.
* Functions like `info()` and `describe()` help understand the dataset's structure, data types, and statistical summary.

## Outcome

After completing Day 4, I can:

* Load CSV files using Pandas
* Display the first and last records of a dataset
* Determine the dataset's dimensions
* View column names and data types
* Generate statistical summaries of numerical data
* Perform an initial exploration of any dataset before analysis
