# Day 2 - NumPy Operations

## Objective

To learn how to access, manipulate, and perform mathematical operations on NumPy arrays efficiently.

---

## Topics Covered

### Indexing

* Accessing individual elements in a NumPy array
* Positive and negative indexing

### Slicing

* Extracting a portion of an array
* Using start, stop, and step values
* Slicing one-dimensional and multi-dimensional arrays

### Mathematical Operations

* Performing element-wise arithmetic operations
* Understanding NumPy's vectorized computations

### Array Reshaping

* Changing the dimensions of an array using `reshape()`

---

## Functions Learned

* `sum()`
* `mean()`
* `max()`
* `min()`
* `reshape()`

---

## Files

* indexing.py
* slicing.py
* mathematical_operations.py
* reshape_arrays.py

---

## Key Learnings

* NumPy allows direct access to array elements using indexing.
* Slicing makes it easy to work with subsets of data.
* Built-in mathematical functions simplify data analysis.
* Reshaping arrays is essential when preparing data for analysis and Machine Learning.

---

## Outcome

After completing Day 2, I can:

* Access array elements using indexing
* Extract data using slicing
* Perform mathematical calculations on arrays
* Reshape arrays into different dimensions
* Use NumPy functions to summarize numerical data efficiently
