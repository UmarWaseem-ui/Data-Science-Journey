# Day 5 - Filtering and Cleaning Data

## Objective

To learn how to select specific data, filter records based on conditions, and handle missing values using Pandas to prepare datasets for analysis.

---

## Topics Covered

### Selecting Columns

* Accessing individual columns
* Selecting multiple columns from a DataFrame

### Data Filtering

* Filtering rows based on conditions
* Retrieving specific records that satisfy given criteria

### Missing Values

* Identifying missing values
* Removing rows containing missing data
* Replacing missing values with appropriate values

---

## Functions Learned

* `df['Age']`
* `df[df['Age'] > 20]`
* `isnull()`
* `dropna()`
* `fillna()`

## Key Learnings

* Selecting specific columns allows focused analysis on relevant data.
* Filtering helps extract records that meet specific conditions.
* Missing values are common in real-world datasets and must be handled before analysis.
* Pandas provides built-in functions to identify, remove, or replace missing data efficiently.
* Data cleaning is a crucial step in the Data Science workflow to ensure accurate analysis and reliable model performance.

---

## Outcome

After completing Day 5, I can:

* Select one or more columns from a DataFrame
* Filter data using conditional statements
* Detect missing values in a dataset
* Remove or replace missing values using appropriate Pandas functions
* Prepare datasets for further analysis and Machine Learning
