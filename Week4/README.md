# Week 4 - SQL and Advanced Excel for Data Analysis

## Overview

This week focuses on developing practical data analysis skills using SQL and Microsoft Excel. The objective is to learn how to retrieve, analyze, summarize, and visualize data using SQL queries and advanced Excel features. These skills are widely used by Data Analysts, Business Analysts, and Data Scientists for working with real-world datasets.

## Skills Gained

By completing this week, I will be able to:

* Write intermediate and advanced SQL queries
* Analyze and summarize data using aggregate functions
* Retrieve data from multiple tables using SQL JOINs
* Apply CASE statements, Subqueries, CTEs, and Window Functions
* Perform advanced data analysis using Microsoft Excel
* Build Pivot Tables and interactive dashboards
* Create KPI reports and business visualizations
* Combine SQL and Excel to solve real-world business problems
