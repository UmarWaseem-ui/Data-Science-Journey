## Key Learnings

* `SELECT` retrieves data from database tables.
* `WHERE` filters records based on specified conditions.
* `ORDER BY` sorts query results in ascending or descending order.
* `LIMIT` restricts the number of records returned.
* `DISTINCT` removes duplicate values from query results.
* Aggregate functions summarize data by calculating totals, averages, minimums, and maximums.
* `GROUP BY` organizes data into groups for analysis.
* `HAVING` filters grouped results based on aggregate conditions.
