To learn how to customize Matplotlib charts for better readability and explore the basics of Seaborn for creating attractive and informative statistical visualizations.

* Why Seaborn is used in Data Science

* Understanding Seaborn's default styles

### Themes

* Applying built-in themes
* Improving the appearance and consistency of visualizations

### Statistical Plots

* Introduction to Seaborn's statistical visualization capabilities
* Simplifying the process of creating informative plots

## Key Learnings

* Well-labeled charts are easier to understand and communicate.
* Titles, axis labels, legends, and grid lines improve the clarity of visualizations.
* Seaborn is built on top of Matplotlib and provides more attractive default styles.
* Seaborn simplifies the creation of statistical graphs with less code.
* Choosing appropriate themes enhances the overall presentation of data.

---

## Outcome

After completing Day 4, I can:

* Customize Matplotlib charts using titles, labels, grids, and legends
* Import and use the Seaborn library
* Apply Seaborn themes to improve chart appearance
* Understand the advantages of Seaborn over basic Matplotlib for statistical visualizations
* Create cleaner and more professional-looking graphs for Data Science projects
