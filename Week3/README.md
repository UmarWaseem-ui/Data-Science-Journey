# Week 3 - Data Visualization with Matplotlib and Seaborn

## Overview

This week focuses on learning Data Visualization, an essential skill in Data Science. The objective is to understand how to present data visually, identify patterns, trends, and relationships, and communicate insights effectively using Python visualization libraries.

---

### Day 1

* Introduction to Data Visualization
* Matplotlib Basics
* Line Charts
* Plot Customization

### Day 2

* Bar Charts
* Pie Charts
* Histograms
* Scatter Plots

### Day 3

* Introduction to Seaborn
* Statistical Visualizations
* Count Plots
* Box Plots

### Day 4

* Heatmaps
* Correlation Matrix
* Pair Plots
* Distribution Plots

### Day 5

* Customizing Visualizations
* Titles
* Labels
* Legends
* Figure Size
* Color Palettes

### Day 6

* Mini Project
* Data Visualization Dashboard
* Exploratory Data Analysis (EDA)
* Presenting Insights Using Charts

---

## Skills Gained

By completing this week, I will be able to:

* Create professional data visualizations using Matplotlib
* Build statistical visualizations using Seaborn
* Customize charts for better readability
* Analyze trends, patterns, and relationships in datasets
* Select appropriate chart types for different kinds of data
* Present data-driven insights visually
