# Day 1 - Introduction to Matplotlib

## Objective

To understand the importance of data visualization in Data Science and learn the basics of creating simple graphs using Matplotlib.

---

## Topics Covered

### Why Data Visualization?

* Understanding the role of visualization in data analysis
* Converting raw data into meaningful insights
* Communicating information through graphs and charts

### Why Graphs are Important

* Simplify complex datasets
* Identify trends and patterns
* Detect outliers and anomalies
* Support better decision-making

### Choosing the Right Chart

* Line Chart for trends over time
* Bar Chart for comparing categories
* Pie Chart for showing proportions
* Scatter Plot for analyzing relationships between variables

### Data Visualization in Data Science

* Exploring data before analysis
* Understanding distributions and patterns
* Supporting Exploratory Data Analysis (EDA)
* Presenting results clearly and effectively

### Creating the First Plot

* Importing Matplotlib
* Creating a basic line graph
* Displaying the graph

* `plt.show()`

## Key Learnings

* Data visualization is a fundamental step in the Data Science workflow.
* Matplotlib is one of the most widely used Python libraries for creating visualizations.
* Different chart types serve different purposes depending on the nature of the data.
* `plt.plot()` is used to create line plots, while `plt.show()` displays the visualization.
