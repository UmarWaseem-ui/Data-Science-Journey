# Day 5 - Heatmaps and Multiple Visualizations

## Objective

To learn how to create heatmaps using Seaborn and understand correlation matrices to identify relationships between variables in a dataset.

---

## Topics Covered

### Heatmaps

* Introduction to heatmaps
* Visualizing data using color intensity
* Creating heatmaps with Seaborn

### Correlation Matrix

* Understanding the correlation matrix
* Measuring relationships between numerical variables
* Interpreting correlation values

### Positive Correlation

* Variables increase or decrease together
* Identifying strong and weak positive relationships
* Visualizing positive correlation through heatmaps

### Negative Correlation

* One variable increases while the other decreases
* Understanding inverse relationships
* Identifying negative correlation using heatmaps

### Multiple Visualizations

* Combining different charts to analyze a dataset
* Comparing insights obtained from various visualization techniques
* Choosing the most appropriate visualization for different types of data

## Key Learnings

* Heatmaps provide an effective way to visualize numerical relationships using colors.
* A correlation matrix summarizes the relationship between all numerical variables in a dataset.
* Positive correlation indicates that two variables move in the same direction.
* Negative correlation indicates that two variables move in opposite directions.
* Heatmaps are commonly used during Exploratory Data Analysis (EDA) to quickly identify important patterns before building Machine Learning models.
