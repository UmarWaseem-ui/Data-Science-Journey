# Day 3 - Histograms and Scatter Plots

## Objective

To learn how to visualize data distributions using histograms and analyze relationships between variables using scatter plots in Matplotlib.

---

## Topics Covered

### Histogram

* Introduction to histograms
* Visualizing the distribution of numerical data
* Understanding the frequency of values within intervals (bins)

### Frequency Distribution

* Understanding how data is distributed
* Identifying the concentration and spread of values
* Detecting patterns such as skewness and outliers

### Continuous Data

* Understanding continuous numerical data
* Representing continuous values using histograms
* Choosing appropriate bins for better visualization

### Scatter Plot

* Introduction to scatter plots
* Plotting two numerical variables
* Visualizing relationships between variables

### Relationship Between Variables

* Observing trends and patterns
* Identifying positive, negative, or no relationship
* Understanding how one variable changes with another

### Correlation

* Introduction to correlation
* Recognizing strong and weak relationships
* Using scatter plots to visually assess correlation

---

## Functions Learned

* `plt.hist()`
* `plt.scatter()`

---

## Files

* Histogram.py

## Key Learnings

* Histograms are used to visualize the frequency distribution of continuous numerical data.
* Scatter plots help identify relationships and trends between two variables.
* Correlation indicates how strongly two variables are related.
* Choosing the correct visualization helps reveal patterns that may not be obvious from raw data alone.

---

## Outcome

After completing Day 3, I can:

* Create histograms using `plt.hist()`
* Create scatter plots using `plt.scatter()`
* Interpret frequency distributions
* Analyze relationships between numerical variables
* Identify basic correlations using data visualizations
