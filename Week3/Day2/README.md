# Day 2 - Bar Charts and Pie Charts

## Objective

To learn how to create bar charts and pie charts using Matplotlib and understand when each chart type should be used to effectively visualize data.

---

## Topics Covered

### Bar Charts

* Introduction to bar charts
* Comparing values across different categories
* Creating bar charts using Matplotlib

### Pie Charts

* Introduction to pie charts
* Representing parts of a whole
* Creating pie charts using Matplotlib

### Choosing the Right Chart

* When to use a bar chart
* When to use a pie chart
* Advantages and limitations of each chart type

---

## Functions Learned

* `plt.bar()`
* `plt.pie()`

---

## Files

* Bar_chart.py
* Pie_chart.py
* Practice_Problems.py

---

## Key Learnings

* Bar charts are ideal for comparing values between different categories.
* Pie charts are useful for showing the proportion of each category within a whole.
* Selecting the appropriate chart type makes data easier to understand and communicate.
* Matplotlib provides simple functions to create clear and informative visualizations.

---

## Outcome

After completing Day 2, I can:

* Create bar charts using `plt.bar()`
* Create pie charts using `plt.pie()`
* Choose the appropriate chart type based on the data
* Present categorical data using effective visualizations
